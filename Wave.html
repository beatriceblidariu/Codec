Wave Format



WAVE (sau pur și simplu Wav) este formatul nativ de fișier utilizat de sistemul de operare Windows pentru stocarea datelor audio digitale. Audio-ul a devenit la fel de important ca imaginile și videoclipurile, motiv pentru care sistemele de operare moderne suportă un format nativ pentru fișiere audio. Popularitatea Windows și cantitatea mare de software disponibil pentru platforma PC au garantat popularitatea formatului Wave.În primul rând, există trei note tehnice:

  • Formatul de fișier Wave este nativ pentru Windows și, prin urmare, rulează pe procesoare Intel care utilizează comanda octet endian mic.
  • Wave prezinta șir de text pentru a specifica o etichetă, notă, și alt informații. Șiruri a căror dimensiune nu este cunoscută în avans sunt stocate în așa-numitul format Pascal în cazul în care primul octet specifică numărul de octeți de text ASCII în șir.
  • WAV (sau . WAV) format de fișier este un caz special de wave în cazul în care nu se utilizează nici o compresie.

O referință recomandată pentru Wave și alte formate de fișiere importante este [Born 95]. Cu toate acestea, este întotdeauna o idee bună pentru a căuta pe Internet pentru referințe noi. Organizarea unui fișier Wave se bazează pe structura riff standard. RIFF vine de la Resource Interchange File Format. Aceasta este baza mai multor formate de fișiere importante.Fișierul este format din bucăți, în cazul în care fiecare bucată începe cu un antet și poate conține subchunks, precum și date. De exemplu, fmt și bucăți de date sunt de fapt subchunks de bucată RIFF. Noi tipuri de bucăți pot fi adăugate în viitor, ceea ce duce la o situație în care software-ul existent nu recunoaște noi bucăți. Astfel, regula generală este de a sări peste orice bucăți de nerecunoscut. Dimensiunea totală a unei bucăți trebuie să fie un număr par de octeți (un multiplu de doi octeți), motiv pentru care o bucată se poate termina cu un octet de căptușeală zero. Prima bucată dintr-un fișier Wave isRIFF. Acesta începe cu riff 4-octet șir, urmată de dimensiunea părții rămase a fișierului (în patru octeți), urmată de string WAVE.
